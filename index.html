License Key Formating

A software licence key is a pattern of numbers and/or letters that is given to an authorized purchaser. When entered by the user during the installation of computer software, the key unlocks a software product and makes it available for use.

It is generated from a private key and validated with its public key. Internal hidden string can also be used in key generation. As in the basic license key customer name and/or company name can be used in key generation and validation.

Given a license key represented as a string s that consists of only alphanumeric characters and dashes. The string is separated into n + 1 groups by n dashes. And also given an integer k This code defines a function reformatLicenseKey that takes the input string s and integer k, removes existing dashes, converts to uppercase, and then reformats the string according to the specified rules
